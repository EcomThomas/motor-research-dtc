# DOC 21b — Banco de Hooks v3.2 (alineado al Spine post-contraste)

Cada hook lleva su **ID de cita origen** real, la inferencia usada, el cluster que explota, la hipótesis que testea y su prioridad de testeo.

**Frame v3.2:** núcleo = madre creyente 45-65 aterrada de morir de hígado graso · gancho = diagnóstico + revertir natural sin cirugía · mecanismo = 'limpiar el hígado' (folk) + bilis/alcachofa (nunca 'detox') · prueba = ultrasonido/kilos/cifra · villano = frituras/azúcar + el DIY casero · hinchazón→hígado = testimonio, no hecho médico (riesgo compliance).
